# Transaction Taxonomy (Informative v0.1)

This taxonomy is intended to be **minimal, extensible, and non-normative**.

## Transaction types (starter set)
- sale
- purchase
- investment
- loan
- lease
- license
- assignment
- merger
- service

## Asset classes (starter set)
- money
- shares
- securities
- immovable_property
- movable_goods
- services
- ip_rights
- hybrid

## Party types (starter set)
- individual
- company
- llp
- partnership
- trust
- govt
- foreign_entity
